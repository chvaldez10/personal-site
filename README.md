# personal-site

### Description

One page personal site for Me.

### Technologies

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn/UI
- Vercel

### Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open the site in your browser at `http://localhost:3000`
